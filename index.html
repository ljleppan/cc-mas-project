Activate Virtual Environment

ljleppan@hp8x-42:~$ source venv/bin/activate

Switch to project root

(venv) ljleppan@hp8x-42:~$ cd pure-rng-decks/

Install dependencies

(venv) ljleppan@hp8x-42:~/pure-rng-decks$ pip install -r requirements.txt

Switch to app directory

(venv) ljleppan@hp8x-42:~/pure-rng-decks$ cd app/

Initialize database

(venv) ljleppan@hp8x-42:~/pure-rng-decks/app$ python manage.py migrate

Modify line 6 of /cards/lib/importer.py so that the variable MASHAPE_KEY contains a valid API key to the API at https://market.mashape.com/omgvamp/hearthstone.

MASHAPE_KEY = "ADD_KEY_HERE"

Start the server

(venv) ljleppan@hp8x-42:~/pure-rng-decks/app$ python manage.py runserver
Performing system checks...

Browse to http://localhost:8000/admin/ and log in

Browse to http://localhost:8000/populate_db. Wait for the program to populate the database from the API. You know the process is complete when you see a page with the text "Done!".

NB: If you get a HTTP Error 403: Forbidden, you either skipped step 9 or provided an incorrect API key.

Browse to http://localhost:8000/admin/cards/card/ and verify that the database got populated.

NB: The next step will take a long time, up to 15 minutes.

Browse to http://localhost:8000/learn and wait for the page to return "Done!".

You are done. To generate a deck, browse to http://localhost:8000/create for a demo of the individual card creation.
